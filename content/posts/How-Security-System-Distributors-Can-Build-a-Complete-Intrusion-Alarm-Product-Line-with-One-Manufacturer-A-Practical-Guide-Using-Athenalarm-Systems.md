## Utökning av produktlinjen: Att bygga ett komplett ekosystem av detektorer

Larmcentraler ensamma skyddar inte lokaler – detektorer skapar det skiktade försvar som definierar professionella system. **Distributörer av säkerhetssystem** som bara erbjuder grundläggande rörelsesensorer missar möjligheter inom perimeter-, miljö- och nödsituationsskydd.

Athenalarm stöder ett fullt spektrum av [trådlösa och trådbundna detektorer](https://athenalarm.com/burglar-alarm/) som fungerar på 433MHz med EV1527-kodning. Kärnkomponenter inkluderar:
- PIR-rörelsedetektorer för volymetriskt skydd inomhus
- Magnetkontakter för dörrar och fönster för övervakning av perimeteringångar
- Glaskross- och vibrationssensorer för högvärdiga tillgångar
- Rök-, gas- och kolmonoxiddetektorer för integration av livssäkerhet
- Panikknappar och vattenläckagesensorer för 24-timmarszoner

Dessa enheter ansluts direkt till AS-9000 eller AS-6000-plattformar utan extra gateways. Distributörer kan paketera skiktade kit: grundläggande för bostäder (PIR + dörrkontakt), detaljhandel (PIR + glaskross + perimeter) eller lager (flera PIR-detektorer + vibration + miljösensorer). Resultatet? Högre genomsnittliga ordervärden och kompletta lösningar som installatörer föredrar framför att blanda varumärken.
